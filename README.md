# 🏪 SportsStore - ASP.NET Core MVC

Đồ án môn **Lập trình Web nâng cao** - Xây dựng website bán hàng thể thao

## 📋 Mô tả

Dự án SportsStore là một ứng dụng web bán hàng thể thao được xây dựng bằng ASP.NET Core 6.0 MVC, sử dụng Entity Framework Core và SQL Server.

## 🚀 Tính năng

- ✅ Hiển thị danh sách sản phẩm với phân trang
- ✅ Lọc sản phẩm theo danh mục
- ✅ Giỏ hàng (Shopping Cart)
- ✅ View Component cho Navigation Menu
- ✅ Responsive UI với Bootstrap 5
- ✅ Entity Framework Core với Code-First Migration

## 🛠️ Công nghệ sử dụng

- **Backend**: ASP.NET Core 6.0 MVC
- **Database**: SQL Server (LocalDB)
- **ORM**: Entity Framework Core 6.0
- **Frontend**: Bootstrap 5, HTML5, CSS3
- **IDE**: Visual Studio 2022
